# Plan por fases (MVP → Pro)

## Fase 0: Estabilización (ahora)
- Definir arquitectura y esquema DB.
- Unificar estilos y layout base.
- Preparar scaffolding (Prisma, auth, i18n).

## MVP (Fase 1)
- Auth + perfiles.
- Campañas: crear/unirse, roles DM/Jugador.
- Personajes: wizard básico (raza, clase, atributos, trasfondo, equipo inicial).
- Compendio: lectura SRD (conjuros/ítems) + filtros básicos.
- Notas de campaña.

## Fase 2: Experiencia completa
- Vista hoja editable en vivo.
- Deshacer/rehacer + historial de cambios.
- Import/export JSON.
- Export PDF (hoja de personaje).
- Traducción y cacheado de SRD (EN/ES).

## Pro (Fase 3)
- Mapas interactivos + handouts.
- Colecciones avanzadas y automatizaciones.
- Colaboración en tiempo real.
- Offline/PWA (si se decide).
